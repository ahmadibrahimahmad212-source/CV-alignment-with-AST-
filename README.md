# CV-JD Alignment Project

This project implements a custom alignment system using Abstract Syntax Trees (AST) to score the alignment between CVs and job descriptions.

## Directory Structure
- `backend/` - Contains the backend server code.
- `frontend/` - Contains the React frontend code.

## Installation

### Backend
1. Navigate to the `backend` directory.
2. Run `npm install` to install dependencies.
3. Start the server with `node server.js`.

### Frontend
1. Navigate to the `frontend` directory.
2. Run `npm install` to install dependencies.
3. Start the development server with `npm start`.

## Usage
Upload a CV and input a job description to get alignment scores.
